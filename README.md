## Usage

The `master` branch is for AWS EC2 deployment in mind. The `develop` branch is for local development.

```
git clone https://github.com/tormund-and-big-lady/Nordstrom-Morgan-proxy.git
cd Nordstrom-Morgan-proxy
npm install
npm run start
```
